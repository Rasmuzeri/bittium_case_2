ListEditor
public
ListEditor
(
JSpinner
spinner)
Construct a
JSpinner
editor that supports displaying
 and editing the value of a
SpinnerListModel
with a
JFormattedTextField
.
This
ListEditor
becomes both a
ChangeListener
on the spinner and a
PropertyChangeListener
on the new
JFormattedTextField
.
Parameters:
spinner
- the spinner whose model
this
editor will monitor
Throws:
IllegalArgumentException
- if the spinners model is not
     an instance of
SpinnerListModel
See Also:
getModel()
SpinnerListModel

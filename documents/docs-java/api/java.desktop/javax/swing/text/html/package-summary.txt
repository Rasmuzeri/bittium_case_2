Package javax.swing.text.html
package
javax.swing.text.html
Provides the class
HTMLEditorKit
and supporting classes for creating
 HTML text editors.

Most of the Swing API is
not
thread safe. For details, see
Concurrency in Swing
,
 a section in
The Java Tutorial
.

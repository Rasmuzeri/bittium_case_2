Interface Rect
public interface
Rect
The
Rect
interface is used to represent any rect value. This
 interface reflects the values in the underlying style property. Hence,
 modifications made to the
CSSPrimitiveValue
objects modify
 the style property.
See also the
Document Object Model (DOM) Level 2 Style Specification
.
Since:
1.4, DOM Level 2

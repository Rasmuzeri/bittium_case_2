Package javax.xml.transform.stax
package
javax.xml.transform.stax
Provides StAX specific transformation classes.
The
StAXSource
class encapsulates a
XMLStreamReader
or
XMLEventReader
and can be used as an input where a
Source
object is accepted.
The
StAXResult
class encapsulates a
XMLStreamWriter
or
XMLEventWriter
and can be used as an output where a
Result
object is accepted.
Since:
1.6
